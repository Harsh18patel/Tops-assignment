HTML elements are the building blocks of HTML pages. With HTML constructs,images and other objects such as interactive forms may be embeded into the renderes page. HTML provides a means to create structured documents by denoting structural semantics for the text as headings, paragraph, lists, link ,quotes,and other items. HTML elements are delineated by tags, written using angle brackets. Tags such as and directly introduce content into the page. other tags such as surround and provide information about document text and may include other tags as sub-elements. Browsers do not display the HTML tags but use them to interpret the content of the page.
